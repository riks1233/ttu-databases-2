PostgreSQL DATA TYPES:
	NUMBERS:
		* smallint
		* integer
		* bigint
	CHARACTERS:
		* character
		* varchar (specified limit)
		* text (unlimited)
	FLOATS:
		* real
		* double precision (not so used, too heavy)
		* decimal and numeric (equal things)
